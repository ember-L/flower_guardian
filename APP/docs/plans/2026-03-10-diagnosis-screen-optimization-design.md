# 病症诊断页面优化设计

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 优化病症诊断页面的视觉设计，使其更现代化、与清新自然主题风格匹配，并添加返回按钮

**Architecture:** 保持现有的三段式结构（初始状态 → 加载中 → 结果展示），添加返回按钮，改进视觉元素

**Tech Stack:** React Native, StyleSheet, lucide-react-native icons

---

## 设计详情

### 1. 头部区域优化
- 增大图标区域至 72x72，添加阴影效果
- 标题字体加粗（fontWeight: 'bold'）
- 副标题改为："AI智能预诊，告别植物杀手"

### 2. 操作按钮优化
- 主按钮（拍照诊断）：渐变绿色背景，添加阴影
- 次按钮（相册选择）：边框样式，添加阴影
- 两个按钮都添加 gap 间距

### 3. 拍摄建议卡片
- 改为胶囊卡片样式
- 添加相机图标装饰
- 增加实用建议条目

### 4. 加载动画
- 增大加载圈（120x120）
- 添加脉冲动画效果

### 5. 诊断结果卡片
- 严重程度使用彩色标签（红/黄/绿）+ 图标
- 原因列表添加图标装饰
- 治疗建议使用编号样式

### 6. 返回按钮
- 左上角，圆形背景
- 与其他子页面保持一致
