# 花园、百科、商城页面顶部现代化设计

**日期**: 2026-03-12
**类型**: UI/UX 优化
**风格**: 渐变 + 毛玻璃

## 1. 设计目标

将三个主要页面（花园、百科、商城）的顶部区域升级为现代化风格：
- 使用粉色渐变背景与 App 主色调协调
- 毛玻璃效果提升层次感
- 保持现有功能不变
- 统一三个页面的视觉语言

## 2. 设计方案

### 2.1 统一视觉规范

| 元素 | 样式规格 |
|------|----------|
| 渐变背景 | LinearGradient: #f46 (primary) → #ff6b88 (primaryLight) |
| 标题 | 白色 #fff, 24-28pt, bold, textShadow |
| 副标题 | 白色 rgba(255,255,255,0.8), 14pt |
| 搜索框 | 毛玻璃: bg rgba(255,255,255,0.25), blur, border-radius: 24 |
| 按钮 | 毛玻璃: bg rgba(255,255,255,0.2), border-radius: 12 |
| 圆角 | 大圆角 16-24pt |

### 2.2 花园页面 (GardenScreen)

**当前状态**:
- 白色背景 + 简单标题行

**优化后**:
```
┌─────────────────────────────┐
│  🌸 我的花园              [+ 添加植物]   │ ← 渐变背景
│  已养护 3 株植物                   │ ← 白色半透明
├─────────────────────────────┤
│  植物卡片列表...                   │
```

**改动点**:
- header 背景改为渐变色
- 标题改为白色 + 阴影
- "添加植物"按钮改为毛玻璃样式

### 2.3 百科页面 (EncyclopediaScreen)

**优化后**:

├─────────────────────────────┤
│  [🔍 搜索植物名称...]          │ ← 毛玻璃搜索框

**改动点**:
- header 背景改为渐变色

## 3. 技术实现

### 3.1 依赖

需要安装 `expo-linear-gradient`:
```bash
npx expo install expo-linear-gradient
```

### 3.2 组件结构

创建可复用的 `GradientHeader` 组件:
```tsx
interface GradientHeaderProps {
  title: string;
  subtitle?: string;
  children?: React.ReactNode;
}
```

### 3.3 修改文件

1. `APP/src/screens/GardenScreen.tsx` - 顶部区域
2. `APP/src/screens/EncyclopediaScreen.tsx` - 顶部区域 + 搜索框
3. `APP/src/screens/StoreScreen.tsx` - 顶部区域 + 搜索框
