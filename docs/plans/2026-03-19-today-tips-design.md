# 今日小贴士功能设计方案

> **设计日期**: 2026-03-19

## 目标

在首页显示"今日小贴士"，通过获取用户当地的气候、温湿度、日照等信息，由 AI 自动生成个性化的植物养护小贴士。

## 数据流程

1. **获取位置** → 使用设备 GPS 获取经纬度
2. **获取天气** → 调用天气 API 获取温湿度、天气状况、空气质量、紫外线等
3. **AI 生成** → 将天气数据发送给 AI，生成个性化小贴士
4. **展示** → 在首页显示天气信息和 AI 生成的小贴士

## 需要的数据

### 天气数据
- 温度（当前/最高/最低）
- 湿度
- 天气状况（晴/雨/阴等）
- 空气质量
- 紫外线指数
- 风速

### AI 生成的小贴士示例
```
🌡️ 今日温度 28°C，湿度 65%
💧 建议早晨浇水，避开高温时段
🌿 根据今日天气，您的绿植需要...
```

## 技术方案

### 后端
- 新增天气 API 端点
- 集成天气服务（和风天气或其他免费 API）
- 调用 AI 生成个性化建议

### 前端
- 使用 react-native-geolocation-service 获取 GPS
- 调用后端 API 获取天气和 AI 建议
- 在首页展示今日小贴士卡片

## 实施范围

1. **后端**: 新增天气 + AI 建议 API
2. **前端**: 首页获取位置和天气，展示小贴士
