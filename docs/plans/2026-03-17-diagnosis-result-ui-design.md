# 病症诊断结果页面优化设计方案

> **Date:** 2026-03-17
> **Goal:** 优化病症诊断结果页面UI，增加图片展示和信息丰富化

---

## 1. 需求概述

优化病症诊断结果页面的用户界面：
1. 显示检测的图片（用户拍摄的原始照片）
2. 丰富诊断结果的信息展示

### 2.2 顶部图片展示区

- **图片显示**: 使用 Image 组件显示 `diagnosisResult.imageUrl`
- **圆角**: borderRadius: 16
- **阴影**: 使用现有 shadows.lg 样式
- **信息标签行**:
  - "识别图片" 标签
  - "在线识别" / "离线识别" 模式标签
  - 置信度百分比（如 "92%"）

### 2.3 严重程度徽章

根据严重程度显示不同颜色：
- 严重 (high): 红色 + AlertCircle 图标
- 中等 (medium): 橙色 + AlertTriangle 图标
- 轻微 (low): 绿色 + Check 图标

## 3. 数据结构

```typescript
interface DiagnosisResult {
  id: string;
  name: string;           // 病害名称
  confidence: number;     // 置信度 0-1
  type: 'insect' | 'disease' | 'physiological' | 'unknown';
  treatment: string;      // 治疗建议
  prevention: string;     // 预防措施
  severity: 'low' | 'medium' | 'high';
  recommendations: {
    immediate: string;    // 紧急处理
    prevention: string;
    severity_level: string;
  };
  imageUrl?: string;      // 识别图片URI ← 新增
}
```
